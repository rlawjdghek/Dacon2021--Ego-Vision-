## Environments Settings
- #### CUDA version >= 11.1
- #### Ubuntu 18.04
```
$ conda env create -n ego --file environment.yml
$ conda activate ego
$ pip install git+https://github.com/ildoonet/pytorch-gradual-warmup-lr.git
```

python main.py
```
